"# JobifyP" 

## Live Demo
(https://jobify-mkag.onrender.com/).

## Features

- Add new job applications  
- Update job status  
- Delete jobs  
- Search and filter jobs  
- Responsive UI built with React  

=========================================

## Tech Stack

- **Frontend:** React  
- **Backend:** Node.js, Express  
- **Database:** MongoDB Atlas  
- **Deployment:** Render
